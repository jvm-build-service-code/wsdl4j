Change History

This document lists the changes from version to version. The most recent version is at the top.

  • Version 1.4
    • build.xml
      • Modified the build script to add version information to the MANIFEST.MF files generated in the created jars.
      • Made naming of generated zip files more consistent.
    • com.ibm.wsdl.util.StringUtils
      • Re-wrote getURL(...) to not open connections when invoked.
    • com.ibm.wsdl.xml.WSDLWriterImpl
      • Changed printDefinition(...) so that <wsdl:documentation> elements are printed prior to <wsdl:import> elements. This was done to respect the order defined in the WSDL schema.
      • Changed all but "utility" methods from private static to protected.
  • Version 1.3
    • Base version. No changes.
